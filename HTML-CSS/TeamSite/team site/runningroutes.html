Running Club

Already ran all the routes on campus? Consider joining Grand Valleys own running club. The Grand Valley running club runs all year round rain or shine. They also have nights where you can hang out and eat pizza with the other members of the club. If you are feeling a bit more competitve you can register for races and run with other club members.
